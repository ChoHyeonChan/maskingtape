# packages/mcp-server — MCP 서버

**담당: 조현찬 (팀장)** · 상태: ⏳ 코어 엔진 v1 이후 시작 (8/11 주 예정)

AI 에이전트가 한국어 데이터를 다루기 전에 거치는 프라이버시 계층.
`packages/core`를 import해서 MCP 도구로 **노출만** 한다 — 탐지 로직을 여기 두지 않는다.

계획된 도구:

| 도구 | 역할 |
|---|---|
| `scan_text` | 텍스트의 개인정보 탐지 리포트 반환 |
| `anonymize_text` | 텍스트 마스킹·가명처리 결과 반환 |
| `anonymize_file` | 파일 단위 배치 비식별화 |
